`bioMEMEPy` is a Python package for motif discovery using the model **OOPS** (One Occurrence Per Sequence).

## Caracteristics

- Extraction of sequences from FASTA file.
- Possibility of finding motifs with DNA, RNA or proteins.
- Nucleotides and proteins already defined within.
- Implementation of a PWM (output).
- **Simple implementation, no external dependencies.**
